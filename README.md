## 📌 Project Overview

**WattWatch** is a Data Analytics project focused on analyzing urban electricity consumption patterns to help build smarter and more sustainable cities.
The project uses **SQL** for data analysis and **Power BI** for interactive dashboard visualization.

This project identifies:

* High energy consumption zones
* Consumer usage patterns
* Monthly energy trends
* Power outage issues
* Estimated billing cost
* Smart city efficiency insights

---

## 🎯 Problem Statement

Cities generate massive electricity data daily. Without analysis, it becomes difficult to identify wastage, outages, or demand trends.

WattWatch solves this by transforming raw smart meter data into actionable insights using analytics and dashboards.

## 📂 Dataset Information

Dataset Name: **SmartCityEnergy.csv**

### Columns Used:

| Column Name        | Description                           |
| ------------------ | ------------------------------------- |
| MeterID            | Unique smart meter ID                 |
| Zone               | North / South / Central               |
| ConsumerType       | Residential / Commercial / Industrial |
| Date               | Reading date                          |
| EnergyConsumed_kWh | Total energy used                     |
| PeakUsage_kWh      | Highest hourly usage                  |
| OutageMinutes      | Power outage duration                 |
| MeterStatus        | Active / Faulty                       |
| TariffRate         | Price per unit                        |

## 💡 Key Insights

* Industrial consumers use the highest electricity
* North zone has highest total usage
* Some zones face higher outage minutes
* Monthly consumption peaks in mid-year periods
* Smart meter faults can impact reliability
